# Implementation of Random Forest Algorithm for Weather Prediction
## AIM:
To write a program to predict daily temperature , PM2.5 pollution level and Energy based on environmental sensor data using Random Forest Algorithm.

## Problem Statement and Dataset
To develop a machine learning model using Random Forest Regressor to predict multiple weather parameters such as temperature, PM2.5 (air pollution), and solar radiation based on historical weather data and engineered features like time and lag variables.

The dataset used is a weather station dataset (weather-station-eee-block_2024_07_13.csv) containing time-based environmental readings.

Main Attributes:

time – Timestamp of the recorded data

tem – Temperature (°C)

pm2_5 – Air pollution level (PM2.5 in µg/m³)

tsr – Solar radiation (W/m²)

hum – Humidity

pressure – Atmospheric pressure

wind_speed – Wind speed

illumination – Light intensity

co2 – Carbon dioxide level

Derived Features:

hour, hour_sin, hour_cos – Time-based cyclical features

tem_lag1, tem_lag2, pm2_5_lag1, pm2_5_lag2, tsr_lag1, tsr_lag2 – Previous time-step (lag) values used for prediction
